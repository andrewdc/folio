# Components

All components live in `src/components/` as `.astro` files.

## Layout & Structure

| Component | Purpose |
|-----------|---------|
| `Nav.astro` | Top navigation bar |
| `Footer.astro` | Site footer |
| `Sidenav.astro` | In-page side navigation for case studies |
| `Band.astro` | Full-width horizontal band section |
| `Section.astro` | Generic content section wrapper |

## Content & Display

| Component | Purpose |
|-----------|---------|
| `Hero.astro` | Page hero with headline and intro |
| `Paths.astro` | Navigation paths / card grid for portfolio sections |
| `MicroPaths.astro` | Smaller variant of Paths |
| `ProjectCard.astro` | Individual project card |
| `Details.astro` | Project detail / metadata block |
| `Challenge.astro` | Case study challenge section |
| `Cta.astro` | Call-to-action block |
| `Metric.astro` | Stat/metric highlight |
| `Stats.astro` | Group of Metric blocks |

## Image Display

| Component | Purpose |
|-----------|---------|
| `ImageGrid.astro` | Multi-image grid layout |
| `ImageRow.astro` | Horizontal row of images |
| `ImageSolo.astro` | Single featured image |
| `Masonry.astro` | Masonry/waterfall image layout (illustration gallery) |
